# NEO ClassBoard

NEO ClassBoard 是一个基于 Qt 6 的 Windows 桌面课表小组件应用，旨在为学生提供实时课程提醒与课表管理功能。

## 功能亮点

- 导入 CSES 格式课表文件
- 今日课程展示、实时倒计时与下节课程提醒
- 换课、替换课程与调休支持
- 课表编辑器：科目管理与时间线设置
- 系统托盘操作：显示/隐藏、设置、编辑、换课
- 迷你模式、透明度、缩放、字体与通知配置
- 课程预备铃与通知音支持

## 核心文件

- `main.cpp`：程序入口，单实例检查、系统托盘与 QML 引擎初始化
- `csesparser.h`：课表解析、配置管理与通知逻辑核心
- `main.qml`：主窗口 UI 逻辑与课程显示
- `Settings.qml`：主视图布局和样式定义
- `settingsdialog.h`：设置窗口实现
- `scheduleeditor.h`：课表编辑器实现
- `classswapdialog.h`：换课对话框实现
- `sample_schedule.yml`：示例课表文件

## 构建要求

- Windows 平台
- CMake 3.16 及以上
- Qt 6.11 及以上

## 构建步骤

1. 在项目根目录创建构建目录，例如 `build`
2. 运行 `cmake ..` 生成构建文件
3. 运行 `cmake --build .` 编译 `ClassBoard`
4. 若可用，构建后会自动尝试运行 `windeployqt` 来打包 Qt 运行时

## 使用说明

- 启动后会显示一个悬浮课表窗口
- 通过系统托盘菜单快速打开设置、课表编辑器或换课界面
- 在设置中导入课表文件并调整显示、通知和外观配置
- 进行换课、调休或单节替换后，应用会保存当天临时课表调整
