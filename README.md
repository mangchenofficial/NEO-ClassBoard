# ClassBoard

ClassBoard 是一个基于 Qt 6 的 Windows 桌面课表小组件应用，主要用于显示和管理学校课程表。

## 项目简介

这个项目通过 Qt Quick + C++ 结合实现：

- 课表数据解析与管理
- 今日课程显示、倒计时与自动更新
- 课程替换/交换功能
- 课表编辑与导入功能
- 系统托盘控制与窗口显示隐藏
- 迷你模式、透明度、缩放等外观设置
- 课程开始前预备铃与通知音支持

## 主要模块

- `main.cpp`：应用入口，创建单实例、托盘菜单、QML 引擎与主窗口。
- `csesparser.h`：课表解析、配置管理、时间与通知逻辑核心。
- `Settings.qml`：应用主界面 QML 视图，负责显示课程信息和通知。
- `settingsdialog.h`：设置对话框窗口，提供应用设置与外观配置。
- `scheduleeditor.h`：课表编辑器窗口，支持科目管理与时间线设置。
- `classswapdialog.h`：换课对话框，支持调课与单节课程替换。
- `sample_schedule.yml`：示例课表数据文件格式。

## 构建要求

- CMake 3.16+
- Qt 6.11+
- Windows 平台

## 构建方式

1. 使用 CMake 生成项目
2. 编译 `ClassBoard` 可执行文件
3. 如果在 Windows 上，构建后会尝试使用 `windeployqt` 部署 Qt 运行时

## 使用说明

- 启动后，应用会显示一个悬浮窗口，展示当天课程与下一节课程信息。
- 系统托盘菜单可快速打开设置、课表编辑器、换课界面，或隐藏/显示窗口。
- 课表可通过导入 CSES 格式文件进行加载。
